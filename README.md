# Viajei

o objetivo do projeto **Viajei** é,comparthilhar das experiências que vivemos, para que outras pessoas também possam usufruir da mesma experiências podem ser sobre qualquer tema (livros,séries,filmes,cozinha,esportes,viagens...).s

>[!WARNING]
> O  **Viajei**,por razões apropriadas,é indicado para maiores de 14 (catorze) anos.**Não recomendamos** que pessoas menor do indicado,tenham acesso à plataforma.

 ## O que será?

O Viajei será uma plataforma em que usuários poderão,individualmante,compartilhar suas exoeriências,e também,recomendar outras tantas para pessoas que acessaram essa plataforma.

> [!IMPORTANT] 
> A plataforma não suporta usuários múltiplos. Isto não é uma rede social virtual. Para compartilhar  suas perripécias, por favor, compartilhe o limk da sua história.
